# WARNING_DELAY: Seconds to wait (default: 5)
#
# HTML Formatting in WARNING_TEXT:
# - Use <strong>text</strong> for bold (gold color)
# - Use <em>text</em> for italic
# - Use <h3>heading</h3> for sub-headings
# - Use <p>paragraph</p> for paragraphs
# - Use <ul><li>item</li></ul> for bullet lists
# - Use <ol><li>item</li></ol> for numbered lists
#
# Examples:
#
# Simple redirect without warning:
# go1;https://google.com
#
# With HTML formatted warning, auto redirect after 5 seconds:
