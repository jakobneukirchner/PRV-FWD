# URL Directory for proeve.net forwarding service
# Format: ID;URL;WARNING_TEXT;WARNING_MODE;WARNING_DELAY
# 
# WARNING_TEXT: Optional warning text to display before redirect
#              Supports HTML formatting AND variables!
# WARNING_MODE: auto (default), wait, button, button-delay
# WARNING_DELAY: Seconds to wait (default: 5)
#
# ===== VARIABLES IN WARNING_TEXT =====
# You can use these variables in WARNING_TEXT:
#
# {countdown}  - Live countdown timer (updates every second)
# {url}        - Target URL
# {delay}      - Initial delay value (doesn't change)
#
# Variables are automatically styled in gold color!
#
# ===== HTML FORMATTING =====
# - <strong>text</strong> for bold (gold color)
# - <em>text</em> for italic
# - <h3>heading</h3> for sub-headings (gold)
# - <p>paragraph</p> for paragraphs
# - <ul><li>item</li></ul> for bullet lists
# - <ol><li>item</li></ol> for numbered lists
#
# ===== EXAMPLES =====
#
# Simple redirect without warning:
# go1;https://google.com
#
# With countdown variable:
# example1;https://example.com;<p>Weiterleitung in <strong>{countdown}</strong> Sekunden zur Website <strong>{url}</strong></p>;auto;8
#
# With all variables:
# example2;https://example.com;<h3>Bitte warten</h3><p>Sie werden in {countdown} Sekunden weitergeleitet.</p><p>Ziel: {url}</p><p>Gesamtwartezeit: {delay} Sekunden</p>;wait;10
#

#
